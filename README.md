# KidsTimer

A kid-friendly visual timeboxing app. Combines an analog clock with colored time-block overlays, a digital countdown timer, and a task list to help children manage their time.

## Features

- Analog clock with real-time hands and colored task overlay
- Digital countdown timer with color-coded border
- Task list with custom colors and durations
- Cute mascot with animations
- LocalStorage persistence
- Responsive design

## Usage

Open `index.html` in a browser or serve with any static file server.
